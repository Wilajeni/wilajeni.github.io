快速选用admonition shortcodes

wilajeni 收录于 技术分享
 2023-04-24  约 307 字   预计阅读 1 分钟 
目录

admonition shortcode 支持 12 种colourful的横幅。有以下命名参数:

  • type [可选] (第一个位置参数)

    admonition 横幅的类型, 默认值是 note.

  • title [可选] (第二个位置参数)

    admonition 横幅的标题, 默认值是 type 参数的值.

  • open [可选] (第三个位置参数)

    横幅内容是否默认展开, 默认值是 true.

例如，按参数名称指定参数：

1{{< admonition type=tip title="This is a tip" open=false >}}
2一个 **技巧** 横幅
3{{< /admonition >}}

或者，按参数位置指定参数（不可和参数名称混用），默认打开横幅：

1{{< admonition tip "这是标题" >}}
2一个 **内容** 
3{{< /admonition >}}

以下为各种颜色对应的类型：

note
一个 注意 横幅
abstract
一个 摘要 横幅
info
一个 信息 横幅
tip
一个 技巧 横幅
success
一个 成功 横幅
question
一个 问题 横幅
warning
一个 警告 横幅
failure
一个 失败 横幅
danger
一个 危险 横幅
bug
一个 Bug 横幅
example
一个 示例 横幅
quote
一个 引用 横幅
